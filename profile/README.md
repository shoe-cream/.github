> ### 주요 기능
**1. 로그인**
- 미리 등록된 계정을 사용하여 사번과 비밀번호 검증을 통해 로그인
  
**2. 사이드바 및 검색창**
- 2뎁스 구조로 페이지 항목이 펼쳐지는 사이드바를 통해 원하는 페이지를 빠르게 이동
- 좌상단 검색창을 통해 페이지를 검색하여 이동
  
**2. 마스터 데이터 관리**
- 제품, 고객사, 제조사, 고객사별 단가 정보를 등록 및 관리
- 추가 버튼을 통해 신규 정보 등록, 행 추가 기능으로 한 번에 여러 데이터 등록 가능
- 표의 개별 셀을 수정하고 행별 체크박스 + 수정 버튼를 통해 수정사항을 최종 반영
- 체크박스와 삭제 버튼을 통해 원하는 데이터 삭제 가능
  
**3. 주문 등록**
- 고객사와 제품 마스터 정보와 해당 고객사 단가 마스터 정보를 불러와 추가 정보를 기입하여 주문 등록
- 각 주문 건에 대한 엑셀 파일 및 견적서 내보내기 가능
- 관리자 계정을 통해 선택한 주문 건 승인/반려 가능
- 각 주문 건에 대한 히스토리(주문 등록, 상태 변경 등) 확인 가능
  
**4. 매출 통계**
- 단위기간별 매출액과 마진율을 시각적으로 확인 가능
